Setup guide
Deploy your agent to Vercel in 5 minutes
This app is deployed on Vercel. Your API key is stored securely as a server environment variable — never exposed to the browser.
Deployment steps
1
Download the project zip
Get the bg-agent.zip file from Claude and unzip it on your computer
2
Create a free Vercel account
Go to vercel.com → Sign up with GitHub (free, no credit card needed)
3
Deploy the project
Vercel dashboard → Add New Project → drag and drop your unzipped folder → Deploy
4
Add your Anthropic API key
In Vercel project settings → Environment Variables → add:

Name: ANTHROPIC_API_KEY
Value: your sk-ant-... key

Then redeploy (Settings → Deployments → Redeploy)
5
Your permanent URL is live
Vercel gives you a URL like https://bg-affiliate-agent.vercel.app — bookmark it and use it from any device, forever.
